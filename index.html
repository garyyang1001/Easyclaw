常見問題

FAQ

什麼是 OpenClaw？

OpenClaw 是一個開源的 AI 助理框架，在 GitHub 上擁有超過 145,000 顆星。它能讓 AI 模型連接到你的通訊平台（Telegram、Discord、WhatsApp 等），成為你的 24/7 私人助理。EasyClaw 幫你省去所有技術設定，一鍵部署完成。

我需要自己的 AI API Key 嗎？

是的。你需要從 Anthropic（Claude）、OpenAI（GPT）或 Google（Gemini）取得 API Key。這確保你完全掌控自己的 AI 使用量和費用。我們的 Web 介面會引導你完成設定。

AI API 大概要花多少錢？

取決於使用量。一般個人使用，每月 AI API 費用約 $5-30 美元。你可以在模型提供商的後台設定用量上限來控制花費。

支援哪些通訊平台？

目前支援 Telegram、Discord、WhatsApp、Slack、Google Chat、Signal、Microsoft Teams 等。LINE 支援即將推出。

我可以自訂 AI 助理的行為嗎？

可以。你可以透過 Web 介面設定自訂 Prompt、指定 AI 的角色與行為，甚至連接外部工具和 API。專業方案提供更多自訂選項。

資料安全嗎？

你的 OpenClaw 實例是完全獨立的，資料不與其他用戶共享。所有通訊都透過加密連線傳輸。你的 API Key 僅存儲在你的專屬實例中。

可以退款嗎？

可以。我們提供 7 天無條件退款保證。如果不滿意，聯繫我們即可全額退款。
